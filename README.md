# NDN-API_SERVER

Using Flask to build a Restful API Server.

## Installation

Install with pip:

```
$ pip install -r requirements.txt
```

## Flask Application Structure 
```
.
|──────app.py
|──────service.py
|──────k8s_client.py
|──────requirements.txt

```
 
## Run Flask
### Run flask for develop
```
$ python app.py
```
In flask, Default port is `5000`
